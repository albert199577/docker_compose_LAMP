# Docker-compose LAMP

## Folder structure

```
project
│  docker-compose.yml
│  README.md
│  
├─mysql
│  └─data
├─php
│      dockerfile
│      
└─public
        index.php
```

## How to create LAMP env?

1. Prepare `php / phpmyadmin / mysql` docker image.
2. Execute Command line ` docker-compose up ` in terminal.
